# Degree Recommendation Tool

An interactive questionnaire that helps prospective Forward College students identify the degree pathway most aligned with their interests and preferences.

## Highlights

- Guides users through a structured, multi-step decision flow
- Loads degree-weighting rules from a configurable CSV scoring matrix
- Maps each response to programme scores and recommends the highest-ranked degree
- Keeps content and scoring logic separate for easier iteration

## Tech Stack

JavaScript, HTML5, CSS3, CSV, and Fetch API.
